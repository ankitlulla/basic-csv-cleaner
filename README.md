# CSV Cleaner

A simple browser-based tool for cleaning CSV files before importing them into Excel, Power BI, or other spreadsheet tools.

## Features

- Upload and preview CSV files
- Remove duplicate rows
- Remove empty rows
- Trim extra whitespace
- Remove empty columns
- Preview cleaned data before downloading
- Export the cleaned CSV

## How to Use

1. Upload a CSV file.
2. Select the cleanup options.
3. Review the cleaned data.
4. Download the updated CSV.

## Privacy

All file processing happens locally in your browser. Your files are never uploaded or stored on a server.
